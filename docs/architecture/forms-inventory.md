# Forms Inventory

This document provides an inventory of all Windows Forms in the PEIMSV3Cs project, detailing their purpose and functional module.

| Filename                          | Purpose                                               | Module                       |
| --------------------------------- | ----------------------------------------------------- | ---------------------------- |
| `AboutBox.cs`                     | Displays application information, version, and credits. | System/Utility               |
| `frmCases.cs`                     | Manages medical or customer support cases.            | Pharmacy/Medical             |
| `frmCasesReport.cs`               | Displays reports for cases.                           | Pharmacy/Medical (Reporting) |
| `frmCategory.cs`                  | Manages product or item categories.                   | Inventory                    |
| `frmCategoryReport.cs`            | Displays reports for categories.                      | Inventory (Reporting)        |
| `frmCountry.cs`                   | Manages country data for addresses, etc.              | System/Utility               |
| `frmCountryReport.cs`             | Displays reports for countries.                       | System/Utility (Reporting)   |
| `frmCustomer.cs`                  | Manages customer information (CRUD operations).       | CRM                          |
| `frmCustomerReport.cs`            | Displays reports for customers.                       | CRM (Reporting)              |
| `frmDepartment.cs`                | Manages company departments.                          | HRM                          |
| `frmDepartmentReport.cs`          | Displays reports for departments.                     | HRM (Reporting)              |
| `frmDisease.cs`                   | Manages information about diseases.                   | Pharmacy/Medical             |
| `frmDiseaseReport.cs`             | Displays reports for diseases.                        | Pharmacy/Medical (Reporting) |
| `frmEmailGmail.cs`                | Provides a UI to send emails via Gmail.               | System/Utility               |
| `frmEmailHotmail.cs`              | Provides a UI to send emails via Hotmail.             | System/Utility               |
| `frmEmailYahoo.cs`                | Provides a UI to send emails via Yahoo.               | System/Utility               |
| `frmEmployee.cs`                  | Manages employee information (CRUD operations).       | HRM                          |
| `frmEmployeeReport.cs`            | Displays reports for employees.                       | HRM (Reporting)              |
| `frmInventoryItem.cs`             | Manages individual inventory items.                   | Inventory                    |
| `frmInventoryItemReport.cs`       | Displays reports for inventory items.                 | Inventory (Reporting)        |
| `frmInvoice.cs`                   | Creates and manages customer invoices.                | CRM / Finance                |
| `frmInvoiceReport.cs`             | Displays reports for invoices.                        | CRM / Finance (Reporting)    |
| `frmLocation.cs`                  | Manages physical locations or warehouses.             | Inventory                    |
| `frmLocationReport.cs`            | Displays reports for locations.                       | Inventory (Reporting)        |
| `frmMain.cs`                      | The main container window for the application.        | System/Utility               |
| `frmMainLogin.cs`                 | The primary login screen for user authentication.     | System/Utility               |
| `frmOrder.cs`                     | Manages customer orders.                              | CRM                          |
| `frmOrderReport.cs`               | Displays reports for orders.                          | CRM (Reporting)              |
| `frmPayrollDetails.cs`            | Manages payroll details for employees.                | HRM / Finance                |
| `frmPayrollReport.cs`             | Displays reports for payroll.                         | HRM / Finance (Reporting)    |
| `frmPharmacyPrescriptionDatabase.cs` | Manages pharmacy prescriptions.                    | Pharmacy/Medical             |
| `frmProduct.cs`                   | Manages product information (CRUD operations).        | Inventory                    |
| `frmProductConsumption.cs`        | Tracks the consumption of products.                   | Inventory                    |
| `frmProductConsumptionReport.cs`  | Displays reports on product consumption.              | Inventory (Reporting)        |
| `frmProductReport.cs`             | Displays reports for products.                        | Inventory (Reporting)        |
| `frmQuickLinks.cs`                | Provides a menu of quick links to other forms.        | System/Utility               |
| `frmRfid.cs`                      | Manages RFID tag information for inventory.           | Inventory                    |
| `frmRfidReport.cs`                | Displays reports for RFID tags.                       | Inventory (Reporting)        |
| `frmShipping.cs`                  | Manages shipping details for orders.                  | CRM                          |
| `frmShippingReport.cs`            | Displays reports for shipping.                        | CRM (Reporting)              |
| `frmSms.cs`                       | Provides a UI to send SMS messages.                   | System/Utility               |
| `frmSplashScreen.cs`              | The splash screen shown on application startup.       | System/Utility               |
| `frmUALogin1.cs`                  | Alternate login form (purpose unclear).               | System/Utility               |
| `frmUALogin2.cs`                  | Alternate login form (purpose unclear).               | System/Utility               |
| `frmUALogin3.cs`                  | Alternate login form (purpose unclear).               | System/Utility               |
| `frmUsers.cs`                     | Manages application users and their permissions.      | System/Utility               |
